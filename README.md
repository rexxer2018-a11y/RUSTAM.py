# Игра "Сборщик монет" на PyGame
Проект демонстрирует применение базовых принципов геймдизайна и архитектуры приложений на Python
Проект, реализованный в рамках практического задания. Игра представляет собой аркадный сборщик монет с системой рекордов.

## Особенности реализации:
* **Объектно-ориентированное программирование (ООП)**:
  - `Player`: логика главного героя и управление.
  - `LeaderboardDB`: взаимодействие с базой данных SQLite3.
  - `CoinCollectorGame`: основной движок игры.
* **База данных**: Использование модуля `sqlite3` для хранения никнеймов и результатов. Реализован вывод Топ-5 игроков.
* **Интерфейс**: 
  - Графический ввод имени пользователя прямо в окне игры.
  - Стартовое окно с инструкциями и финальное окно с таблицей лидеров.
* **Геймплей**: Управление стрелками клавиатуры. Сам сборщик(Синий кружок)
* Цель — собрать 10 монет(Желтые кружки) , избегая столкновения с врагами (Красные кружки).

## Инструкция по запуску:
1. Установите Python предпочтительно 3.8.X
   но можно и новее
2. Установите библиотеку Pygame:
   pip install -r requirements.txt
3. Запустите основной файл игры:
   ```bash
   python RUSTAM.py
   ```
* Имеется   также  запакованный под виндовс EXE - файл:
  (pyinstaller --onefile --noconsole RUSTAM.py),   чтобы можно было запускать игру щелчком мыши под виндовс.
